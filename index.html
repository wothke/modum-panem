Click mouse or press 'enter' to show this window. List of other controls:
  • 'space' to toggle between scripted and interactive flight path.
  • 'ctrl' toggles 'look around' mode while on scripted flight path
  • 'q', 'a', ←, ↑, →, ↓ keys are used to steer while in interactive mode (alternatively mouse dragging and mouse wheel can be used).
  • 'F11' to toggle full screen mode
  • use hash URL to select whatever SoundCloud music you want to play (default music by https://www.wrightandbastard.com/)

Use Chrome for correct rendering!
